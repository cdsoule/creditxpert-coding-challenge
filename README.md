# CreditXpert Coding Challenges

## Requirements
- Python3 (tested on 3.12.2)

## Set up environment (macOS/Linux)
1. Clone repo
- `$ git clone <repo>`
2. Enter directory
- `$ cd <repo_name>`
3. Create virtual environment
- `$ python -m venv venv`
4. Activate venv
- `$ . venv/bin/acitvate`
5. Install requirements
- `$ python -m pip install -r requirements.txt`

## Local deployment
1. Run server
- `$ python manage.py runserver`
